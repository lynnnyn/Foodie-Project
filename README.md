# Foodie-Project
a website for food share

# About our website
## Technology stack
* Python 3.6
* Flask
* jQuery
* ES6
* MySQL
* cPanel

## Installation
1. Get the code. Clone this git repository and check out the latest release:
git clone https://github.com/lynnnyn/Foodie-Project.git
    cd Foodie-Project
    git checkout latest

2. run main.py

3. website will run on http://0.0.0.0:2000
